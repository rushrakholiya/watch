Add product without user frontend access!

== Description ==

When plugin enabled you can let your user to submit a product in frontend way.
Page option will appear in your WordPress backend menu, and you can find shortcode to display frontend form.
You can add simple product in this free version of plugin. If user are not logged they can't see the form.

https://vimeo.com/195978963

PREMIUM FEATURES of add product frontend:

Add **variable** product
**Show/hide field**
View, edit and delete product for **each user**

For more information about PREMIUM VERSION, visit official page <a href="http://www.nsthemes.com/product/frontend-add-product/" target="_blank">NsThemes Add product frontend</a>.

*We test our plugins with many other. If you find incompatibility with one specific plugin, please inform us at <a href="mailto:info@nsthemes.com">info@nsthemes.com</a>.*


== Installation ==

1. Upload the entire 'ns-frontend-add-product' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'Add Prod' and start to customize it.
